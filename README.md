# River-Class

Models a river with properties and water quality constants.
Class Design:
Constants: POLLUTED = 1, CLEAN = 2
Fields: riverName, waterQuality, length, drainageBasinArea
Static counter: numberOfRiver
No-arg and parameterized constructors
Example:
plain
Input:
Changjiang
Huanghe

Output:
name: Changjiang
water quality: 2
length: 10.0
drainage basin area: 100.0
number of rivers: 1
River{riverName='Changjiang', waterQuality=2, length=10.0, drainageBasinArea=100.0}
